### 5.3 友邦保险（AIA）对比——为什么中国险企P/EV低于1x？

| 维度 | 友邦保险(AIA) | 中国内资险企平均 |
|------|:-----------:|:--------------:|
| **P/EV** | **1.50 - 1.56x** | **0.46 - 0.73x（A股）** |
| NBV Margin | 50%+ | 16-29% |
| 代理人质量 | 精英化、高产能 | 仍在提质增效中 |
| 利差损风险 | 极低 | 市场持续担忧 |
| 地理分布 | 亚太多元化 | 高度集中于中国内地 |
| 负债久期管理 | 匹配度高 | 久期缺口待缩小 |

**中国寿险P/EV低于1x的核心原因（华创证券、东方证券等研报总结）：**

1. **EV置信度受损（最核心）**：低利率环境下，市场担忧EV中隐含的长期投资收益率假设（通常5%）不再成立。广发证券测算当前P/EV隐含的市场对未来投资收益率预期极为悲观（国寿3.0%、平安2.3%、太保1.8%、新华2.8%），均低于预测稳态总投资收益率4.1%。

2. **"利差损"担忧**：10年期国债收益率降至1.7-1.8%，存量保单负债成本约3%，利差损压力压制估值。

3. **净投资收益率逼近负债成本**：2025年上市险企加权净投资收益率已逼近约3%的刚性负债成本线。

4. **广义回报率持续下行**：2018年以来广义回报率（综合投资收益率-VIF打平收益率）均值持续下行，与PEV估值下行高度吻合。

### 5.4 积极信号——估值修复催化剂

| 利差损风险缓解 | 头部险企"利差损"风险基本出清（方正证券判断） |

## 七、核心结论

### 估值锚点总结

```
新华保险 P/EV（2026E）：A股 0.63-0.67x，H股 ~0.48x
行业平均 P/EV：A股 0.61x（十年分位 3.1%）
友邦保险 P/EV：1.50-1.56x（品质溢价 2.5-3倍）

新华保险在板块内 P/EV 略高于行业均值（0.65x vs 0.61x），
但绝对值仍处于"破净"（P/EV < 1x）状态。
```

### 多空因素对照

| 看多因素 | 看空因素 |
|---------|---------|
| NBV连续三年高增（+25%/+107%/+57%） | 2025年高基数下2026年NBV增速或放缓至10-15% |
| 银保渠道弯道超车（NBV贡献53.6%） | 银保margin（13.9%）仍远低于个险 |
| 2026Q1利润增速领跑行业（+10.5%） | Q1投资收益率仅2.1%，公允价值变动亏损209亿 |
| 分红险转型突破（Q4占比77%） | 分红险长期盈利能力待验证 |
| P/EV处于历史中位，不算贵 | 但也不处于历史极端低位 |
| H股股息率6.33%，类债券配置价值 | 外资大行普遍看空（美银/汇丰/高盛均卖出） |
| 权益弹性大（牛市收益弹性强） | 权益弹性大（熊市亏损弹性也大） |

### 投资含义

1. **P/EV锚点**：0.65x 2026E P/EV是当前的核心估值锚。东方证券认为合理估值为0.8x，对应目标价81.22元（+36%上行空间）。
